# 题目背景

在遥远的提瓦特大陆，弥哈尤部落正准备发动一场决定生死存亡的远征。为了赢得这场战争，大先知决定举行一场古老的献祭仪式，以祈求元神的祝福。

按照传统习俗，部落需要在出征前从军队中挑选一名士兵进行献祭。作为部落的首席军事策略家，需要评估献祭每一名士兵后对军队整体力量的影响，以便大先知能做出最明智的决策。

# 题目描述

部落拥有一支由 $n$ 名士兵组成的军队。第 $i$ 名士兵的力量值为 $a_i$。一支军队的总力量定义为其中所有士兵力量值的乘积。

计算对于从 $1$ 到 $n$ 的每一名士兵 $i$，依次计算当其被献祭后，由剩余 $n-1$ 名士兵组成的军队的总力量。

由于计算结果可能很大，所有计算出的总力量都需要对 $998244353$ 取模。

# 输入格式

第一行输入一个整数 $n\ (2\le n\le{10}^7)$，代表士兵的数量。

第二行输入 $n$ 个整数 $a_1, a_2, \dots, a_n\ (0\le a_i\le100)$，代表每名士兵的力量值。

# 输出格式

仅一行包含 $n$ 个整数，以空格分隔。

其中第 $i$ 个整数表示献祭第 $i$ 名士兵后，军队中剩余士兵的总力量对 $998244353$ 取模的结果。

# 测试样例

```input1
4
3 1 4 2
```

```output1
8 24 6 12
```

```input2
4
1 2 0 4
```

```output2
0 0 8 0
```

        $$
        10\bmod3=1
        $$
        对于本题，模有一个显而易见的性质：
        $$
        {(a_1a_2\cdots a_k)}\bmod N = {(({a_1}\bmod N)({a_2}\bmod N)\cdots({a_k}\bmod N))}\bmod N
		$$
        例如
        $$
        \begin{aligned}
        {(4\cdot8\cdot7)}\bmod 3&={(({4}\bmod3)\cdot({8}\bmod3)\cdot({7}\bmod3))}\bmod3 \\
        {224}\bmod3&={(1\cdot2\cdot1)}\bmod3 \\
        2&=2
        \end{aligned}
        $$
        这意味着你可以在做乘法之前可以随意（对同一个数）取模。
